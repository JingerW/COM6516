The sheffield package

The sheffield package contains three classes for simplifying input, output and graphical display in Java, called EasyReader, EasyWriter and EasyGraphics.

To install the package, create a subdirectory called sheffield in the directory which you will use to store your java programs (e.g. H:\myjava\sheffield). Download the following zip file into the sheffield directory by clicking here:

sheffield.zip

To uncompress the archive, use ZipCentral on the Windows NT machines or type the following at the UNIX prompt (make sure you are currently in the sheffield directory):

unzip sheffield.zip

If all has gone well, you should see some java source files, the corresponding class files and documentation in HTML format, which can be read in a web browser.
